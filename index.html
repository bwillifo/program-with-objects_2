Inheritance

One topic that I found relevent to help programmers is the idea of inheritance. To arrange the hierarchy that represents the "is-a-type-of" relationship. To allow for all data and methods avaialble to the parent class to also appear in teh child calsss within the same names. This allows for the easy re-use of the same procedures and data definitions to mirror the relationship in a intuitive way. Such as the Billy Cyrus to Mily Cyrus example, we can pass to the Child the last_name for example.

In the below example, I am using an Child/Parent Inheritance and while I can seek to have the CHild inherint the parent attributes, I have set up a subclass that will over ride the parent attributes. In this case I should get a parent with the last name of WIlliford and with brown eyes, while the Child will get last name of Williford, eyes are blue and 100000 toys. To build a subclass to override the parent class is called Method Override

class Parent():
def __init__(self, last_name, eye_color):
print("Parent Constructor Called")
self.last_name = last_name
self.eye_color = eye_color

def show_info(self):
print("Last Name - "+self.last_name)
print("Eye Color - "+self.eye_color)

class Child(Parent):
def __init__(self, last_name,eye_color, number_of_toys):
print("Child Constructor Called")
Parent.__init__(self, last_name, eye_color)
self.number_of_toys = number_of_toys

def show_info(self):
print("Last Name - "+self.last_name)
print("Eye Color - "+self.eye_color)
print("Number of toys - "+str(self.number_of_toys))

brent_williford = Parent("Williford", "brown")
brent_williford.show_info()
keegan_williford = Child("Williford","blue", 1000000)
keegan_williford.show_info()
